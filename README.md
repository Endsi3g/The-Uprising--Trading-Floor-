# 🚀 The Uprising Trading Floor

**The Uprising Trading Floor** is an elite, AI-powered multi-engine trading suite designed for extreme efficiency, high-performance execution, and "12-year-old simple" user experience. It integrates three of the world's most powerful open-source trading engines into a unified Next.js dashboard.

---

## 🌍 Vision / Vision (FR)

**English**: To democratize institutional-grade trading tools using advanced AI self-improvement loops and a simplified, intuitive interface.

**Français**: Démocratiser les outils de trading de qualité institutionnelle grâce à des boucles d'auto-amélioration par l'IA et une interface simplifiée et intuitive.

---

## 🏗️ Core Architecture

The system orchestrates dynamic trading across multiple engines:

| Engine | Role | Use Case |
| :--- | :--- | :--- |
| **Hummingbot** | High-Frequency & Connectivity | Arbitrage, Market Making, Gateway execution. |
| **Freqtrade** | Algorithmic & Backtesting | Trend following, DCA strategies, comprehensive backtesting. |
| **OctoBot** | Community & Grid Trading | Multi-currency grid strategies, community-driven signals. |
| **Trading AI Suite** | Intelligence Layer | Sentiment analysis, risk management, AI strategy optimization. |

---

## 🖥️ Unified Dashboard

The dashboard provides a "Command Center" view:
- **Multi-Layout Charting**: Single or grid views for intensive monitoring.
- **Bot Manager**: Centralized control for starting, stopping, and monitoring all connected engines.
- **AI Sentinel**: Real-time risk scoring and "circuit breaker" alerts.

## 🛠️ Quick Start

### Prerequisites
- Python 3.10+
- Node.js 18+
- Docker & Docker Compose
- `Ollama` (Running DeepSeek or Llama3)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Endsi3g/The-Uprising--Trading-Floor-.git
   cd hummingbot
   ```
2. Set up the environment:
   ```bash
   ./bin/install
   ```
3. Start the system:
   ```bash
   ./bin/start
   ```
